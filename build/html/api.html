The EpochDialog class¶

The EpochDialog class represents a dialog box that allows the user to select a historical epoch from a list of options.

The UnitDialog class¶

The UnitDialog class represents a dialog box that allows the user to select a unit of measurement from a list of options.

The CSVMapper class represents the main window of the application.

You can also sort data by a specific column. The EpochDialog and UnitDialog classes provide dialogs for selecting historical epochs and units of measurement. The show_epoch_dialog and show_unit_dialog functions are called when the user double-clicks a cell in the “Epoch” or “Unit” column of the data table. These functions display historical era and unit dialog boxes, respectively, and allow the user to select an era or unit from the combobox. The on_google_sheet_action_triggered, save_csv and save_google functions manage saving data to Google Sheets or CSV files. The update_relations function manages the updating of the relationships between the stratigraphic units present in the data table. This function reads the CSV file or Google Sheets spreadsheet, updates the relationships between the stratigraphic units, and returns a DataFrame containing the updated rows. The update_relationships_in_dataframe function updates the relationships between stratigraphic units present in a specific DataFrame. The show_changes and ask_to_save_changes functions are called when the user makes changes to the data table and needs to decide whether to save the changes to a CSV file. The variables data_table, epochs_df, unit_df, spreadsheet_id, df, data_fields, df2, and data_fields2 are used as containers for the data. (“index era”) and “Unit” (unit of measurement) columns. The “index era” column contains the names of the historical epochs, and the “Unit” column contains the names of the units of measurement.
